# External Dependency Audit

Public web search and local filesystem search were performed for original-author confusion matrices/exact code, can-train-and-test-v1.5, can-sleuth, and official event boundaries. The original can-train-and-test paper and local original dataset are available; exact public confusion matrices, exact benchmark code, v1.5, and official event boundaries were not found in the accessible sources checked here.

This is blocker evidence for exact-alignment claims, not a reason to drop the paper. The paper should state metric ambiguity and corrected benchmark evidence, and include this audit in the appendix.
